# dotfiles

## 導入方法
```
git clone https://github.com/punkrou404/dotfiles.git
cd dotfiles
. ./init.sh
```

## vimrcの構成

```
.vimrc
.vim
|--config
| |--dein
| | |--plugins.toml
| | |--plugins.vim
| | |--plugins_lazy.toml
| |--init
| | |--base.vim
| | |--display.vim
| | |--input.vim
| | |--keymap.vim
```

## zshrcの構成

```
.zshrc
.zsh
|--config
| |--init
| | |--base.sh
| | |--pronpt.sh

```
